Peak Lab

Peak Lab is my consulting practice, focused on helping small trades and service businesses — landscapers, electricians, contractors — make the jump from solopreneur chaos to something that actually runs like a business.

Most of my clients are great at their craft but drowning in the operational side: no CRM, no lead tracking, quoting jobs from memory, chasing invoices manually. I come in and fix the infrastructure — systems, software, and the processes that connect them.

02
Dev project · Health · Data

Peak Lab Health

A personal health tracking dashboard that pulls data from Strava, Garmin, Hevy, and MacroFactor into one place — and uses AI to surface patterns across all of them.

The problem it solves: I was tracking everything but understanding nothing. Sleep data in Garmin, runs in Strava, lifts in Hevy, food in MacroFactor — none of it talked to each other. This app connects the dots and asks the question: what actually affects how I feel and perform?

Built in React with Vite. Currently in early development — visualisations and data correlation logic are underway. This is also my main learning project as a beginner developer, so the build is as much about understanding the code as shipping the product.

React JavaScript Strava API Garmin API Data viz AI analysis
